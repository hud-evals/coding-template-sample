# Coding Template Sample Repository

This is a sample repository used by the [coding-template](https://github.com/hud-evals/coding-template) as a fallback when no custom `REPO_URL` is provided when building through hud.ai. The task that uses this repository can be found in the `tasks/basic.py` file.

## Branches

- `sample_bug_fix_baseline` - Web server with a bug
- `sample_bug_fix_test` - Baseline + test that exposes the bug
- `sample_bug_fix_golden` - Baseline + the fix
